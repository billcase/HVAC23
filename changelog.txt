Changelog / Update Notes:
- Flange & Single-line Pipe Tool: Updated.
- Ductwork Tool: Optimized for faster drawing. Added automatic duct segmenting into blocks (CVD) to facilitate automated quantity take-offs (QTO).
- Ductline (DDline): Added functionality for automatic duct segmentation based on standard manufacturing sizes.
- DIM Tool: Updated. Optimized for creating Dimension Styles that comply with A3 printing standards across various scales.
- QDB: Added "Quick Dimension at Block Point" feature for rapid dimensioning at the basepoint of selected blocks.
- PPR & PVC Fittings: Updated support for QQ1, QQ2, and QQ3 commands. These commands now automatically identify pipe types (Steel, PVC, PPR) and insert the corresponding fittings.
- TML (Mline Counter): Updated with additional configuration options.
- New Commands Added: TKB: Ccount att block and other..., BlockUnits: Manage block unit settings, SCI, WI1: Create Wipeout,KHBV..)
- Excel Integration: Updated the import/export functionality for tables between Excel and AutoCAD.
- DD3 Command: Added (similar to DD1 and DD2, but uses DIM). Generates statistics/take-offs for lines based on picked points.
- EW Command: Added versatile text editing functionality. Allows direct editing of text, even within blocks and attribute blocks.
- Update-Check: Added a command to check for available software updates.
- And various other minor improvements.
